# Changelog

## [0.1.0] - 2026-06-04

### Added
- 初始版本：用户与 Agent 之间的翻译层 skill
- **SKILL.md**：核心理念、使用说明、7核心方法、S/A/B分级访谈、15维度概述
- **7个核心方法** (M0-M7)：
  - M0 结构化反馈引导（四要素：在哪/做了什么/什么现象/期望什么）
  - M1 联想与补全（展开用户没意识到的维度）
  - M2 渐进式路线图（拆 Phase，拒绝一把梭）
  - M3 双向约束（决策原子性约束 + 选项中性化检查）
  - M4 参考优先协议（问用户→搜索→找不到透明告知）
  - M5 混合状态门 + 反推动计数器（信号词=动手，模糊=弹框，禁止=不动）
  - M6 能力边界声明（项目/代码/AI三层边界 + 低数据量语言标记 + 跨语言污染防护）
  - M7 实现后审查与债务管理（/review调用 + 重构安全协议六步骤）
- **15维度分级访谈**：S级4项（隐私/边界/确认阈值/环境路径）、A级9项、B级4项
- **对话哲学**：苏格拉底式探询 + 半隐式方向确认 + 隐式信号解读
- **references/interview-guide.md**：完整访谈指南 + "我不懂"处理协议 + 决策原子性约束
- **references/gate-protocol.md**：状态门完整逻辑 + 反推动计数器机制
- **references/review-protocol.md**：审查触发 + 重构安全协议 + 硬性/软性评判框架
- **references/output-template.md**：CLAUDE.md + CLAUDE.d/ 生成模板
- **references/defaults.md**：43项默认参数（代码/日志/环境工具/版本控制/交互层）
- **硬门机制**：日志写入成为 IMPLEMENT→PLAN 状态切换前置条件
- 环境上下文感知：启动时自动判断运行环境（终端/VSCode/受限）
